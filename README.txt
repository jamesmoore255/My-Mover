My-Mover Project

A simple application for the web which informs the user of recent NYTimes articles and Wikipedia articles relating to the destination of which they want to move. It also includes of a google maps picture of the location in the background. It was an app to help utilise AJAX requests and work with API's.


Clone the repo onto your computer, and then simply open the index.html file, you will then be able to use the app.

Built with HTML5, CSS3, JavaScript and JQUERY.
